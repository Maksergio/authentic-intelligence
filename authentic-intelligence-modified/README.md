## About

**The Authentic Intelligence** is a holistic ethical governance approach that positions human judgement, community context, and ethical accountability as indispensable complements to artificial intelligence.

Created by **Serge MAKOUNDOU**, CEO of Culture Tour Ltd.

### The Three Pillars

| Framework | Expansion | Purpose |
|-----------|-----------|---------|
| **CNM** | Context – Nexus – Modelling | Meta-framework for human-centric AI decisions |
| **EPI** | ETHICAL Proximity Index | Seven-dimension ethical scoring instrument |
| **DEMAND** | Data – Ethics – Matrix – Agent – Nexus – Dynamics | Operational decision cycle |

## Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sergemakoundou/authentic-intelligence.git

# 2. Enter the folder
cd authentic-intelligence

# 3. Install dependencies
npm install

# 4. Start the app
npm run dev
```

Open your browser at **http://localhost:3000/authentic-intelligence/**
